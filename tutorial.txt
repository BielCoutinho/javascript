GIT e GITHUB

Comandos básicos do git:
git config --global user.name "Seu nome"
git config --global user.email "seuemail@email.com"
git clone url_do_repositório (clonar um repositório)
git add . (adiciona ao repositório local)
git commit -m "tutotial" (documentar as alterações)
git push (enviar ao repositório)
git pull (sicroniza as informações do repositório) no pc
git status (verificar o status do repositório local)
